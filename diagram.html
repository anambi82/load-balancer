Flow Diagram

Overview

The flow diagram shows the main loop of the load balancer. The process begins with obtaining the user input for the number of servers and how long the balancer will be running. Then during each iteration (clock cycle but will likely be a loop) the IP address will be checked to see whether it is able to enter the queue or not and then the scaling conditions get checked to see what needs to be done. The loop will only end once the time limit mathecs the user specified input, and then from there logs will be made and the program will clean up and end.
