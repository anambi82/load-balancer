Detailed Description

Holds all configuration parameters for the load balancer simulation.

Parameters can be loaded from a key=value config file or set to built-in defaults. The LoadBalancer reads this object at startup to determine initial server count, scaling thresholds, request probabilities, and blocked IP ranges.
