Detailed Description

Writes simulation events to a log file and optionally to the console.

Each log entry is prefixed with a zero-padded clock cycle number. The class also tracks aggregate statistics (servers created/deleted, requests processed/blocked) and writes a formatted summary at the end of the simulation.
